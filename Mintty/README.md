Mintty

Configuration
-------------

[Wiki](https://github.com/mintty/mintty/wiki/Tips)

Configuration folders (in order of being read):

* /etc/minttyrc
* $APPDATA/mintty/config
* ~/.config/mintty/config
* ~/.minttyrc

Resource sub-folders:
* themes
* sounds
* lang
